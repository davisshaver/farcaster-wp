Farcaster WP connects your WordPress site to Farcaster. 

== Description ==

Farcaster WP makes it easy to setup [Farcaster frames](https://docs.farcaster.xyz/developers/frames/v2/) meta tags for your WordPress site.

- Enable frames for your WordPress site
- Set button text, splash screen background color, and splash screen icon
- Validate your Farcaster manifest config

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/farcaster-wp` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure the plugin through the 'Farcaster Settings' screen in the WordPress admin.
